== Description ==

Custom plugin built by Southern Tide Media to pull data from ZohoCRM and show it on a Google Map

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `find-a-provider` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
